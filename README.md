# Customer-Segmentation-and-Predictive-Analytics-Project

This project applies Machine Learning (ML) techniques to solve two business problems: predictive analytics for customer renewal behavior and customer segmentation based on product preferences. Decision tree modeling and hierarchical clustering are used in R to generate insights and support data-driven decision-making.

## Tools & Technologies
R Programming, RStudio, rpart, rattle, tidyverse, cluster, ggplot2, kableExtra, knitr, scales, Power BI.

## Methodology
Data preprocessing, model building, and evaluation were performed in R. A decision tree model was developed for classification, and performance was evaluated using training and testing datasets. Hierarchical clustering with Euclidean distance was applied for segmentation, followed by cluster profiling using visualizations and summary statistics.

## Key Outcomes
-Decision tree model achieved 62% training accuracy and 51% test accuracy, indicating mild overfitting
-Key predictors of renewal included length of relationship (LOR), spend, and age
-Pruned model improved generalisation with 52% test accuracy
-Three customer segments were identified using hierarchical clustering
-Clustering produced a mean silhouette score of 0.255, indicating weak but interpretable structure
-Clear differences in product preferences (D1, D3, D5) were observed across clusters
-Segmentation insights support targeted marketing strategies aimed at improving customer retention and potential revenue growth
